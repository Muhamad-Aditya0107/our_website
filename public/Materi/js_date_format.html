JavaScript Date Format

JavaScript menyediakan beberapa cara untuk memformat tanggal dan waktu. Anda dapat menggunakan metode seperti toLocaleDateString(), toLocaleTimeString(), serta menggunakan pustaka seperti Intl.DateTimeFormat untuk memformat tanggal dalam berbagai cara sesuai dengan kebutuhan lokal atau format yang diinginkan.

Menggunakan toLocaleDateString() dan toLocaleTimeString()
  • toLocaleDateString(): Mengonversi tanggal menjadi string yang terformat sesuai dengan lokal tertentu.
  • toLocaleTimeString(): Mengonversi waktu menjadi string terformat sesuai dengan lokal tertentu.
  • Kedua metode ini memberikan fleksibilitas dalam memformat tanggal dan waktu sesuai dengan standar lokal, seperti format tanggal di Amerika Serikat (MM/DD/YYYY) atau di Eropa (DD/MM/YYYY).
const currentDate = new Date();

console.log(currentDate.toLocaleDateString('en-US')); // Output: 12/24/2024
console.log(currentDate.toLocaleDateString('en-GB')); // Output: 24/12/2024

console.log(currentDate.toLocaleTimeString('en-US')); // Output: 10:30:45 AM
console.log(currentDate.toLocaleTimeString('fr-FR')); // Output: 10:30:45
Menggunakan Intl.DateTimeFormat()
  • Intl.DateTimeFormat() adalah konstruktor yang memungkinkan Anda memformat tanggal dan waktu menggunakan opsi format yang lebih terperinci.
  • Dengan menggunakan Intl.DateTimeFormat, Anda bisa menentukan format tanggal dan waktu dengan kontrol lebih lanjut seperti menentukan apakah Anda ingin menampilkan hari dalam seminggu, bulan dalam angka atau nama bulan, dan banyak lagi.
const currentDate = new Date();

const formatter = new Intl.DateTimeFormat('en-US', {
year: 'numeric',
month: 'long',
day: 'numeric'
});
console.log(formatter.format(currentDate)); // Output: December 24, 2024

const timeFormatter = new Intl.DateTimeFormat('en-US', {
hour: 'numeric',
minute: 'numeric',
second: 'numeric'
});
console.log(timeFormatter.format(currentDate)); // Output: 10:30:45 AM
Custom Date Formatting
  • Untuk memformat tanggal dengan cara yang lebih custom, Anda bisa mengekstrak bagian-bagian dari objek Date seperti tahun, bulan, dan hari, lalu merangkainya secara manual.
  • Ini memberikan fleksibilitas penuh dalam menentukan urutan elemen dan format lainnya, seperti menambahkan leading zeros atau format 24 jam.
const currentDate = new Date();

let day = currentDate.getDate();
let month = currentDate.getMonth() + 1; // Month is 0-based
let year = currentDate.getFullYear();
let formattedDate = `${month < 10 ? '0' + month : month}/${day < 10 ? '0' + day : day}/${year}`;
console.log(formattedDate); // Output: 12/24/2024

let hours = currentDate.getHours();
let minutes = currentDate.getMinutes();
let seconds = currentDate.getSeconds();
let formattedTime = `${hours < 10 ? '0' + hours : hours}:${minutes < 10 ? '0' + minutes : minutes}:${seconds < 10 ? '0' + seconds : seconds}`;
console.log(formattedTime); // Output: 10:30:45
Kesimpulan
  • JavaScript menyediakan berbagai cara untuk memformat tanggal dan waktu dengan menggunakan metode seperti toLocaleDateString() dan toLocaleTimeString().
  • Dengan menggunakan Intl.DateTimeFormat, Anda dapat menyesuaikan format tanggal dan waktu sesuai dengan kebutuhan lokal dan keinginan pengguna.
  • Jika Anda membutuhkan format tanggal dan waktu yang lebih khusus, Anda dapat mengekstrak bagian-bagian dari objek Date dan menyusunnya sesuai format yang diinginkan.
