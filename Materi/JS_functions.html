JavaScript Functions

Fungsi di JavaScript digunakan untuk mengelompokkan kode yang dapat dipanggil berulang kali untuk melakukan tugas tertentu. Fungsi dapat menerima input (parameter) dan mengembalikan output (nilai).

Mendeklarasikan Fungsi

Fungsi dapat dideklarasikan dengan menggunakan kata kunci function, diikuti dengan nama fungsi, parameter (opsional), dan blok kode yang dikelilingi oleh tanda kurung kurawal.

Sintaksis Fungsi

function namaFungsi(parameter1, parameter2) {
    // kode yang akan dijalankan
}

Contoh Fungsi

function greet(name) {
    return "Hello, " + name;
}
let message = greet("John");
console.log(message); // Output: Hello, John

Parameter dan Argument

Fungsi dapat menerima parameter yang memungkinkan Anda untuk mengirim data ke dalam fungsi tersebut. Ketika fungsi dipanggil, data yang dikirim disebut argument.

Contoh Parameter dan Argument

function sum(a, b) {
    return a + b;
}
let result = sum(5, 3); // Argument 5 dan 3
console.log(result); // Output: 8

Fungsi dengan Nilai Kembali (Return)

Fungsi dapat mengembalikan nilai menggunakan kata kunci return. Ini memungkinkan fungsi untuk mengirimkan hasil setelah selesai dijalankan.

Contoh Fungsi dengan Return

function multiply(a, b) {
    return a * b;
}
let result = multiply(4, 5);
console.log(result); // Output: 20

Fungsi Anonim (Anonymous Function)

Fungsi anonim adalah fungsi yang tidak memiliki nama. Fungsi ini biasanya digunakan sebagai callback function atau sebagai argumen dalam fungsi lain.

Contoh Fungsi Anonim

let add = function(a, b) {
    return a + b;
};
let result = add(2, 3);
console.log(result); // Output: 5

Fungsi Panah (Arrow Function)

Fungsi panah adalah cara singkat untuk mendeklarasikan fungsi di JavaScript. Sintaksisnya lebih ringkas dibandingkan dengan fungsi tradisional.

Contoh Fungsi Panah

let subtract = (a, b) => a - b;
let result = subtract(10, 5);
console.log(result); // Output: 5

Fungsi Sebagai Parameter

Fungsi juga dapat diteruskan sebagai parameter ke fungsi lain. Ini memungkinkan kita untuk menulis kode yang lebih modular dan fleksibel.

Contoh Fungsi sebagai Parameter

function greeting(name, callback) {
    return callback(name);
}
function sayHello(name) {
    return "Hello, " + name;
}
let message = greeting("Alice", sayHello);
console.log(message); // Output: Hello, Alice

Kesimpulan

Fungsi adalah bagian yang sangat penting dalam JavaScript. Fungsi memungkinkan kita untuk mengelompokkan kode, memisahkan logika, dan menghindari duplikasi. Ada beberapa cara untuk mendeklarasikan fungsi, seperti fungsi tradisional, fungsi anonim, dan fungsi panah.
