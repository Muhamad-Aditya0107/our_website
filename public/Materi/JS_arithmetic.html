JavaScript Arithmetic

Operator aritmatika digunakan untuk melakukan operasi matematika pada angka (literal atau variabel).

Daftar Operator Aritmatika

Operator Deskripsi
+ Penjumlahan
- Pengurangan
* Perkalian
** Pemangkatan
/ Pembagian
% Sisa Bagi (Modulus)
++ Increment
-- Decrement

Contoh Operasi Aritmatika

Penjumlahan

let x = 5;
let y = 2;
let z = x + y; // z bernilai 7

Pengurangan

let x = 5;
let y = 2;
let z = x - y; // z bernilai 3

Perkalian

let x = 5;
let y = 2;
let z = x * y; // z bernilai 10

Pembagian

let x = 5;
let y = 2;
let z = x / y; // z bernilai 2.5

Modulus

Operator % mengembalikan sisa hasil pembagian:

let x = 5;
let y = 2;
let z = x % y; // z bernilai 1

Pemangkatan

Operator ** digunakan untuk pemangkatan:

let x = 5;
let z = x ** 2; // z bernilai 25

Increment

Operator ++ menambah nilai variabel sebesar 1:

let x = 5;
x++;
let z = x; // z bernilai 6

Decrement

Operator -- mengurangi nilai variabel sebesar 1:

let x = 5;
x--; 
let z = x; // z bernilai 4

Prioritas Operator

Prioritas operator menentukan urutan eksekusi dalam operasi aritmatika:

let x = 100 + 50 * 3; // Hasilnya adalah 250 (perkalian lebih dulu)

Gunakan tanda kurung untuk mengubah prioritas:

let x = (100 + 50) * 3; // Hasilnya adalah 450
