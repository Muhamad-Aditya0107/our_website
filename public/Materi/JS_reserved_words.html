JavaScript Reserved Words

Reserved words adalah kata-kata yang sudah memiliki makna khusus dalam JavaScript dan tidak dapat digunakan sebagai nama variabel, fungsi, atau objek.

Kategori Reserved Words
  • Keywords
    Kata-kata yang memiliki makna khusus dalam JavaScript, seperti:
    • if, else, for, while, return, function
  • Data Types
    Kata-kata yang digunakan untuk mendeklarasikan tipe data:
    • null, true, false, undefined
  • Control Flow
    Kata-kata untuk kontrol alur dan pengelolaan kode:
    • break, continue, switch, case, default
  • Declarations
    Kata-kata yang digunakan untuk mendeklarasikan variabel dan fungsi:
    • let, const, var, function
  • Object and Array Methods
    Kata-kata terkait objek dan array, seperti:
    • this, super
  • Class and Module Keywords
    Kata-kata yang digunakan dalam sistem kelas dan modul ES6:
    • class, extends, import, export
Daftar Reserved Words di JavaScript
JavaScript memiliki daftar kata-kata yang sudah dipakai oleh mesin JavaScript dan tidak bisa digunakan sembarangan, seperti:
  • abstract, await, enum, finally, interface, private, public, static, dan lain-lain.
Kesimpulan
  • Menggunakan reserved words dalam JavaScript sebagai nama variabel atau fungsi dapat menyebabkan error karena kata-kata ini sudah memiliki makna khusus dalam bahasa tersebut. Oleh karena itu, pastikan untuk tidak menggunakan reserved words dalam kode Anda.
