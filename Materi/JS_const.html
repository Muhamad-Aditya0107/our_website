JavaScript const

Kata kunci const diperkenalkan dalam ES6 (2015) dan digunakan untuk mendeklarasikan variabel yang nilai atau referensinya tidak dapat diubah.

Properti Variabel const

  • Variabel yang dideklarasikan dengan const tidak dapat dideklarasikan ulang.
  • Variabel yang dideklarasikan dengan const tidak dapat diubah nilainya (reassigned).
  • Variabel yang dideklarasikan dengan const memiliki cakupan blok.

Contoh: Tidak Dapat Diubah

const PI = 3.141592653589793; 
PI = 3.14;      // Akan memberikan error 
PI = PI + 10;   // Akan memberikan error

Harus Diberikan Nilai Saat Dideklarasikan

Variabel const harus diberikan nilai saat dideklarasikan:

Benar:

const PI = 3.14159265359;

Salah:

const PI; 
PI = 3.14159265359; // Error

Kapan Menggunakan const?

Gunakan const ketika nilai variabel tidak akan diubah. Contohnya:

  • Array baru
  • Objek baru
  • Fungsi baru
  • RegExp baru

Array Konstan

Anda dapat mengubah elemen dalam array konstan:

const cars = ["Saab", "Volvo", "BMW"]; 
cars[0] = "Toyota"; // Mengubah elemen 
cars.push("Audi");  // Menambah elemen baru

Tetapi Anda tidak dapat mengganti seluruh array:

const cars = ["Saab", "Volvo", "BMW"]; 
cars = ["Toyota", "Volvo", "Audi"]; // Error

Objek Konstan

Anda dapat mengubah properti dalam objek konstan:

const car = {type: "Fiat", model: "500", color: "white"}; 
car.color = "red";      // Mengubah properti 
car.owner = "Johnson";  // Menambah properti baru

Tetapi Anda tidak dapat mengganti seluruh objek:

const car = {type: "Fiat", model: "500", color: "white"}; 
car = {type: "Volvo", model: "EX60", color: "red"}; // Error

Perbandingan var, let, dan const

Berikut perbandingan:

Fitur var let const
Cakupan Global Blok Blok
Dapat Dideklarasikan Ulang Ya Tidak Tidak
Dapat Diubah Nilainya Ya Ya Tidak

Kesimpulan

Gunakan const untuk mendeklarasikan variabel yang nilainya tidak akan berubah, seperti array atau objek yang sifatnya konstan.
