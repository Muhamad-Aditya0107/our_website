JavaScript Date Get Methods

JavaScript menyediakan berbagai metode untuk mengambil bagian-bagian dari objek Date, seperti tahun, bulan, hari, jam, menit, detik, dan milidetik. Metode-metode ini memungkinkan Anda untuk bekerja dengan tanggal dan waktu secara lebih terperinci.

Metode `getFullYear()`, `getMonth()`, dan `getDate()`
  • getFullYear(): Mengambil tahun dalam format 4 digit (misalnya 2024).
  • getMonth(): Mengambil bulan dalam format 0-11 (dimulai dari 0 untuk Januari, 1 untuk Februari, dan seterusnya).
  • getDate(): Mengambil hari dalam bulan, dengan rentang 1 hingga 31.
const currentDate = new Date();

let year = currentDate.getFullYear();
let month = currentDate.getMonth();
let day = currentDate.getDate();

console.log(year); // Output: 2024
console.log(month); // Output: 11 (Desember)
console.log(day); // Output: 24
Metode `getDay()`, `getHours()`, `getMinutes()`, `getSeconds()`, dan `getMilliseconds()`
  • getDay(): Mengambil hari dalam minggu dengan rentang 0-6 (0 untuk Minggu, 1 untuk Senin, dan seterusnya).
  • getHours(): Mengambil jam dalam format 24 jam (rentang 0-23).
  • getMinutes(): Mengambil menit dalam format 0-59.
  • getSeconds(): Mengambil detik dalam format 0-59.
  • getMilliseconds(): Mengambil milidetik dalam format 0-999.
const currentDate = new Date();

let dayOfWeek = currentDate.getDay();
let hours = currentDate.getHours();
let minutes = currentDate.getMinutes();
let seconds = currentDate.getSeconds();
let milliseconds = currentDate.getMilliseconds();

console.log(dayOfWeek); // Output: 2 (Selasa)
console.log(hours); // Output: 10
console.log(minutes); // Output: 30
console.log(seconds); // Output: 45
console.log(milliseconds); // Output: 100
Menggunakan `getTime()`
  • getTime(): Mengambil waktu dalam milidetik sejak 1 Januari 1970 (epoch time).
  • Metode ini berguna untuk perhitungan dan perbandingan tanggal, karena mengembalikan nilai angka yang dapat diolah lebih lanjut.
const currentDate = new Date();

let timeInMilliseconds = currentDate.getTime();

console.log(timeInMilliseconds); // Output: 1712304567890 (contoh)
Kesimpulan
  • Metode getFullYear(), getMonth(), dan getDate() memungkinkan Anda untuk mengambil bagian tanggal yang lebih spesifik seperti tahun, bulan, dan hari dalam bulan.
  • Metode getDay(), getHours(), getMinutes(), getSeconds(), dan getMilliseconds() digunakan untuk mengambil hari dalam minggu, waktu, dan detik, memberikan informasi lebih mendalam tentang waktu.
  • Dengan getTime(), Anda dapat memperoleh waktu dalam milidetik sejak epoch time, yang memungkinkan perbandingan tanggal yang lebih mudah.
