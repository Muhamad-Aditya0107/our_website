JavaScript Number Properties

JavaScript menyediakan properti untuk angka yang memungkinkan Anda mengakses nilai-nilai batasan, seperti nilai tak terbatas (Infinity), selisih terkecil antara angka (Number.EPSILON), dan nilai batas maksimum atau minimum dari angka.

1. Number.MAX_VALUE

Nilai angka terbesar yang dapat direpresentasikan dalam JavaScript. Ini adalah angka yang paling besar sebelum nilai tak terbatas (Infinity).

console.log(Number.MAX_VALUE); // Output: 1.7976931348623157e+308
2. Number.MIN_VALUE

Nilai angka terkecil yang dapat direpresentasikan dalam JavaScript. Ini adalah angka positif yang sangat kecil, bukan nol.

console.log(Number.MIN_VALUE); // Output: 5e-324
3. Number.POSITIVE_INFINITY

Nilai positif tak terbatas. Ini sering muncul dalam operasi yang tidak terhingga, seperti pembagian dengan nol.

console.log(1 / 0); // Output: Infinity
4. Number.NEGATIVE_INFINITY

Nilai negatif tak terbatas. Ini sering muncul dalam operasi yang tidak terhingga dengan bilangan negatif, seperti pembagian -1 dengan nol.

console.log(-1 / 0); // Output: -Infinity
5. Number.NaN

Nilai yang menunjukkan "Not a Number". Ini dihasilkan ketika operasi yang tidak valid dilakukan, seperti perkalian antara string dan angka.

console.log("abc" * 10); // Output: NaN
6. Number.EPSILON

Selisih terkecil antara 1 dan angka berikutnya yang dapat direpresentasikan. Ini membantu untuk mengatasi masalah presisi dalam perhitungan.

console.log(Number.EPSILON); // Output: 2.220446049250313e-16
7. Number.MAX_SAFE_INTEGER

Batas maksimum integer yang aman dalam JavaScript. Ini adalah angka terbesar yang dapat diwakili dengan presisi penuh tanpa kehilangan informasi.

console.log(Number.MAX_SAFE_INTEGER); // Output: 9007199254740991
8. Number.MIN_SAFE_INTEGER

Batas minimum integer yang aman dalam JavaScript. Ini adalah angka terkecil yang dapat diwakili dengan presisi penuh.

console.log(Number.MIN_SAFE_INTEGER); // Output: -9007199254740991
Kesimpulan
  • JavaScript memiliki berbagai properti untuk angka yang digunakan untuk mengakses nilai-nilai batasan dan properti khusus lainnya seperti Infinity dan NaN.
  • Properti seperti *Number.MAX_VALUE, **Number.MIN_VALUE*, dan lainnya memungkinkan kita untuk bekerja dengan nilai-nilai ekstrem dalam perhitungan.
