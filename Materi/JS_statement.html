JavaScript Statements

Contoh

let x, y, z;    // Pernyataan 1
x = 5;          // Pernyataan 2
y = 6;          // Pernyataan 3
z = x + y;      // Pernyataan 4

Program JavaScript

Sebuah program komputer adalah daftar "instruksi" yang akan "dieksekusi" oleh komputer.

Dalam bahasa pemrograman, instruksi-instruksi ini disebut sebagai pernyataan (*statements*).

Sebuah program JavaScript adalah daftar pernyataan pemrograman.

Dalam HTML, program JavaScript dieksekusi oleh peramban web.

Pernyataan JavaScript

Pernyataan JavaScript terdiri dari:

  • Nilai
  • Operator
  • Ekspresi
  • Kata Kunci
  • Komentar

Pernyataan ini memberi tahu peramban untuk menulis "Hello Dolly." di dalam elemen HTML dengan id="demo": 

document.getElementById("demo").innerHTML = "Hello Dolly.";

Sebagian besar program JavaScript berisi banyak pernyataan JavaScript.

Pernyataan-pernyataan tersebut dieksekusi satu per satu, dalam urutan yang sama seperti yang ditulis.

Program JavaScript (dan pernyataan JavaScript) sering disebut kode JavaScript.

Titik Koma (;)

Titik koma memisahkan pernyataan-pernyataan JavaScript.

Tambahkan titik koma di akhir setiap pernyataan yang dapat dieksekusi: 

let a, b, c;  // Deklarasi 3 variabel
a = 5;        // Berikan nilai 5 ke a
b = 6;        // Berikan nilai 6 ke b
c = a + b;    // Berikan hasil penjumlahan a dan b ke c

Saat dipisahkan dengan titik koma, beberapa pernyataan dalam satu baris diperbolehkan: 

a = 5; b = 6; c = a + b;

Di web, Anda mungkin melihat contoh tanpa titik koma. Menambahkan titik koma di akhir pernyataan tidak diwajibkan, tetapi sangat disarankan.

Spasi Putih dalam JavaScript

JavaScript mengabaikan banyak spasi. Anda dapat menambahkan spasi putih ke skrip Anda untuk membuatnya lebih mudah dibaca.

Baris berikut setara: 

let person = "Hege";
let person="Hege";

Praktik yang baik adalah menambahkan spasi di sekitar operator (= + - * /): 

let x = y + z;

Blok Kode JavaScript

Pernyataan JavaScript dapat dikelompokkan bersama dalam blok kode, di dalam kurung kurawal {...}.

Tujuan dari blok kode adalah untuk menentukan pernyataan yang akan dieksekusi bersama.

Salah satu tempat Anda menemukan pernyataan yang dikelompokkan bersama dalam blok adalah di fungsi JavaScript: 

function myFunction() {
  document.getElementById("demo1").innerHTML = "Hello Dolly!";
  document.getElementById("demo2").innerHTML = "How are you?";
}

Kata Kunci JavaScript

Pernyataan JavaScript sering kali dimulai dengan sebuah kata kunci untuk mengidentifikasi tindakan JavaScript yang akan dilakukan.

Berikut adalah daftar beberapa kata kunci yang akan Anda pelajari dalam tutorial ini:

  • var - Mendeklarasikan sebuah variabel
  • let - Mendeklarasikan variabel dalam blok
  • const - Mendeklarasikan konstanta dalam blok
  • if - Menandai blok pernyataan untuk dieksekusi berdasarkan kondisi
  • switch - Menandai blok pernyataan untuk dieksekusi pada beberapa kasus
  • for - Menandai blok pernyataan untuk dieksekusi dalam sebuah perulangan
  • function - Mendeklarasikan sebuah fungsi
  • return - Keluar dari fungsi
  • try - Mengimplementasikan penanganan kesalahan pada blok pernyataan
