JavaScript Classes

Kelas (class) di JavaScript adalah templat untuk membuat objek. Kelas memperkenalkan cara yang lebih bersih dan terstruktur untuk membuat objek dan menangani pewarisan dibandingkan dengan metode berbasis prototipe.

Membuat Kelas
  • Kelas didefinisikan menggunakan kata kunci class. Di dalam kelas, terdapat metode khusus bernama constructor yang digunakan untuk menginisialisasi properti objek.
    class Person {
    constructor(name, age) {
    this.name = name;
    this.age = age;
    }
    greet() {
    return `Hi, my name is ${this.name} and I am ${this.age} years old.`;
    }
    }

    // Membuat objek dari kelas
    const person1 = new Person("Alice", 30);
    console.log(person1.greet()); // Output: Hi, my name is Alice and I am 30 years old.
  • Metode dalam Kelas
    • Metode Biasa: Didefinisikan langsung di dalam kelas.
    • Static Methods: Metode yang hanya dapat dipanggil pada kelas itu sendiri, bukan pada instance.
    class MathUtil {
    static add(a, b) {
    return a + b;
    }
    }

    console.log(MathUtil.add(5, 7)); // Output: 12
  • Pewarisan (Inheritance)
    • Kelas dapat mewarisi properti dan metode dari kelas lain menggunakan kata kunci extends.
    class Animal {
    constructor(name) {
    this.name = name;
    }

    speak() {
    return `${this.name} makes a noise.`;
    }
    }

    class Dog extends Animal {
    speak() {
    return `${this.name} barks.`;
    }
    }

    const dog = new Dog("Rex");
    console.log(dog.speak()); // Output: Rex barks.
Kesimpulan
  • Kelas adalah cara modern untuk membuat objek di JavaScript.
  • Mendukung constructor, metode, pewarisan, getter, dan setter.
  • Mempermudah pengelolaan kode dan pewarisan dibandingkan pendekatan berbasis prototipe.
