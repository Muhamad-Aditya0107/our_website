JavaScript Events

Events adalah aksi atau kejadian yang terjadi di browser, seperti klik, hover, input, atau halaman selesai dimuat. Dengan JavaScript, Anda dapat merespons events untuk membuat halaman lebih interaktif.

Apa itu Events?

Events adalah aksi atau kejadian, seperti klik tombol atau pergerakan kursor. JavaScript digunakan untuk menangkap dan merespons events tersebut.

Contoh Events Umum
  • onclick: Saat elemen diklik.
  • onmouseover: Saat kursor berada di atas elemen.
  • onmouseout: Saat kursor meninggalkan elemen.
  • onchange: Saat nilai input berubah.
  • onload: Saat halaman selesai dimuat.
Menangani Events

JavaScript menyediakan beberapa cara untuk menangani events:

1. Inline HTML

Tambahkan langsung di elemen HTML:

<button onclick="alert('Button clicked!')">Click Me</button>

2. Menggunakan Properti HTML DOM

Atur event handler melalui JavaScript:

document.getElementById("myBtn").onclick = function() {"{"}
  alert("Button clicked!");
{"}"};

3. Menggunakan Event Listener

Pendekatan modern dan fleksibel:

document.getElementById("myBtn").addEventListener("click", function() {"{"}
  alert("Button clicked!");
{"}"});
Event Object

Event Object memberikan informasi tentang event yang terjadi. Contoh properti:

  • event.type: Jenis event (klik, input, dsb.).
  • event.target: Elemen yang memicu event.
Kesimpulan
  • JavaScript Events memungkinkan interaksi dinamis dengan pengguna.
  • Anda dapat menangani events menggunakan berbagai metode, seperti inline HTML, DOM properties, atau Event Listeners.
  • Events membuat halaman lebih interaktif dan responsif.
