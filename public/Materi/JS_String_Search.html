JavaScript String Search

String search adalah metode untuk mencari teks atau pola dalam sebuah string. JavaScript menyediakan berbagai metode untuk melakukan pencarian ini secara fleksibel.

Apa itu String Search?

String search adalah metode yang digunakan untuk mencari teks atau pola dalam sebuah string.

Metode String Search

Berikut adalah metode-metode utama yang dapat digunakan untuk mencari string:

1. indexOf(searchValue, start)

Mengembalikan indeks kemunculan pertama dari teks yang dicari. Mengembalikan -1 jika tidak ditemukan.

let text = "Hello World";
console.log(text.indexOf("World")); // 6

2. lastIndexOf(searchValue, start)

Mengembalikan indeks kemunculan terakhir dari teks yang dicari.

let text = "Hello World World";
console.log(text.lastIndexOf("World")); // 12

3. search(pattern)

Mencari string menggunakan pola teks atau ekspresi reguler. Mengembalikan indeks dari kecocokan pertama.

let text = "Hello World";
console.log(text.search("World")); // 6

4. match(pattern)

Mengembalikan semua kecocokan berdasarkan ekspresi reguler.

let text = "abc abc";
console.log(text.match(/abc/g)); // ["abc", "abc"]

5. includes(searchValue, start)

Memeriksa apakah string mengandung teks tertentu. Mengembalikan true atau false.

let text = "Hello World";
console.log(text.includes("World")); // true

6. startsWith(searchValue, start)

Memeriksa apakah string dimulai dengan teks tertentu.

let text = "Hello World";
console.log(text.startsWith("Hello")); // true

7. endsWith(searchValue, length)

Memeriksa apakah string berakhir dengan teks tertentu.

let text = "Hello World";
console.log(text.endsWith("World")); // true
Kesimpulan
  • JavaScript menyediakan berbagai metode untuk mencari string, seperti indexOf, includes, dan match.
  • Metode seperti match dengan ekspresi reguler sangat fleksibel untuk pencarian pola yang kompleks.
  • Metode-metode ini memungkinkan pencarian teks yang lebih efektif dan sesuai dengan kebutuhan aplikasi Anda.
