JavaScript Errors

JavaScript Errors terjadi saat kode menemukan masalah yang mencegah eksekusi normal. Kesalahan ini bisa berasal dari sintaks, runtime, atau logika dalam kode.

Jenis-Jenis Error di JavaScript
  • Syntax Error
    • Kesalahan pada sintaks kode yang mencegah eksekusi program.
    console.log("Hello World; // Lupa menutup tanda kutip
  • Reference Error
    • Terjadi ketika variabel yang tidak dideklarasikan diakses.
    console.log(x); // Error: x is not defined
  • Type Error
    • Kesalahan yang terjadi ketika operasi dilakukan pada tipe data yang salah.
    let num = 10;
    num.toUpperCase(); // Error: toUpperCase is not a function
  • Range Error
    • Terjadi jika nilai di luar rentang yang valid diberikan.
    let num = new Array(-1); // Error: Invalid array length
  • URI Error
    • Terjadi saat fungsi URI seperti decodeURIComponent() digunakan dengan nilai yang salah.
    decodeURIComponent('%'); // Error: URI malformed
Menangani Error dengan try...catch
  • Gunakan blok try...catch untuk menangani error dan mencegah program berhenti berjalan.
    try {

    console.log(x); // ReferenceError
    } catch (error) {
    console.error("An error occurred:", error.message);
    }
Throwing Errors
  • Gunakan throw untuk melempar error secara manual.
    let age = -5;
    if (age < 0) throw new Error("Age cannot be negative");
Debugging dan Pencegahan Error
  • Periksa kode secara manual untuk kesalahan sintaks.
  • Gunakan console.log() untuk memeriksa nilai variabel.
  • Gunakan debugger di browser untuk melacak kesalahan.
Kesimpulan
  • Kesalahan di JavaScript dapat dikategorikan menjadi syntax, reference, type, range, dan URI errors. Gunakan try...catch untuk menangani error dan lakukan debugging untuk mengidentifikasi serta memperbaiki masalah.
